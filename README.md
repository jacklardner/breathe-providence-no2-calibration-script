# breathe-providence-no2-calibration-script

This application is a streamlined script that pulls NO2 data from the Breathe
Providence low cost air sensor network. Users can change the labeled parameters
in the main.py file to download and calibrate NO2 data, which will then be
delivered as a CSV. A report is also planned to be printed.

This README will be updated as the project progresses. An API key for the
quantAQ data is required.
